# Pre-Implementation Be Engineer Consultation (REQUIRED)

Before implementing any new subsystem or major feature, consult the be-systems-engineer agent with reference to the Be Newsletter archive and Haiku source (/Users/lane/src/haiku/).

The consultation must cover:
1. How did Be/Haiku implement the equivalent?
2. What newsletter articles discuss its design rationale?
3. What worked, what didn't, what would they do differently?
4. How should we adapt given pane's architecture?

This is not optional. It grounds implementation in actual engineering experience.
